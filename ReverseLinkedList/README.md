# \#206. Reverse Linked List

Reverse a Singly Linked List.  A linked list could be reversed either iteraitvely or recursively.

```
head => 1 -> 2 -> 3 -> null
head => 3 -> 2 -> 1 -> null

Inputs:
<ListNode 1> LL

Output:
<ListNode 3> LL

Edge Cases:
n = length
single node input

Assumptions:
always at least size one

```
